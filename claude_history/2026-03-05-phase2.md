# Phase 2 기능 추가

## 작업 요약

Phase 2 로드맵 항목 3개를 모두 구현 완료하고 문서를 업데이트함:
- 정렬 옵션: `s` 키로 date/project/messages 순환
- 세션 통계: `i` 키로 통계 팝업 (총 세션 수, 도구별, 상위 10 프로젝트)
- 설정 파일: `~/.config/asm/config.toml` 또는 `~/.asm.toml` 지원
- 새 의존성: `toml = "0.8"`

## 변경된 파일

- `src/config.rs` — 신규, 설정 파일 로드
- `src/main.rs` — mod config 추가, Config 로드
- `src/app.rs` — Config 수용, Mode::Stats, 's'/'i' 키바인딩
- `src/session.rs` — 커스텀 경로 파라미터, file_path 필드 추가, delete 리팩토링
- `src/tree.rs` — SortMode enum, SessionStats, cycle_sort, stats()
- `src/ui.rs` — 통계 팝업, 정렬 모드 표시, 상태바 업데이트
- `Cargo.toml` — toml 의존성 추가
- `CLAUDE.md` — 아키텍처, 기술 스택, 기능 상세, 로드맵 업데이트
- `README.md` — Features, Keybindings, Dependencies, Configuration 섹션 추가
